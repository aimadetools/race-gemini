# Product Hunt Launch Screenshots - Descriptions

This document outlines the ideal screenshots for the Product Hunt launch. The aim is to visually convey the core value proposition and key features of LocalLeads.

## Screenshot 1: Homepage / Landing Page

**Focus:** The main landing page (`index.html`) after a user signs up or logs in.
**Key Elements to Highlight:**
*   Clear, concise headline immediately communicating the core value (e.g., "Generate Local SEO Pages in Minutes").
*   Prominent Call-to-Action (CTA) button leading to the "Generate" page.
*   Visually appealing, modern design.
*   Optional: A subtle background image or hero section demonstrating local businesses or growth.
*   If logged in, show user's name/profile in the top right.

## Screenshot 2: SEO Page Generator - Input Form

**Focus:** The input form where users configure their SEO pages (`generate.html`).
**Key Elements to Highlight:**
*   Input fields for "Business Name", "Services", "Towns".
*   "Enable AI Copy" toggle.
*   "AI Style" dropdown (e.g., Professional, Casual).
*   "Primary Color" selector.
*   Preview of estimated credits needed for generation.
*   "Generate Pages" button.
*   Clean, intuitive form layout.

## Screenshot 3: Generated SEO Page - Example

**Focus:** A fully generated SEO page, showcasing the output.
**Key Elements to Highlight:**
*   A sample generated page URL (e.g., `localseogen.com/plumber-in-london-johns-plumbing.html`).
*   The business name, service, and town prominently displayed.
*   AI-generated marketing copy.
*   Evidence of schema markup (e.g., using a schema validator tool's screenshot, or just showing the rendered content which implies schema).
*   Clean, responsive design.
*   Contact information (telephone, address).

## Screenshot 4: User Dashboard - Credits & Usage

**Focus:** The user's personal dashboard (`dashboard.html`).
**Key Elements to Highlight:**
*   Current credit balance.
*   Recent generation history (e.g., a list of recently generated pages).
*   Links to "Buy Credits" or "View All Pages".
*   User-friendly interface showing account overview.

**Focus:** The agency dashboard (`agency-dashboard.html`) or a view demonstrating white-label capabilities.
**Key Elements to Highlight:**
