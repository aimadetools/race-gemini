Custom Domain Configuration

Map your own custom domain (like seo.yourplumberdomain.com) to your generated landing pages instead of using our default URLs.

Point a CNAME record in your DNS settings to localseogen.com.

DNS Configuration Guide

Select your domain provider below for step-by-step CNAME configuration instructions:

  1. Log in to your Cloudflare Dashboard and choose your domain.
  2. Navigate to the DNS settings tab on the left sidebar.
  3. Click + Add Record to create a new entry.
  4. Set Type to CNAME.
  5. Set Name (Host) to your desired prefix (e.g., seo or @ for root).
  6. Set Target (Value) to localseogen.com.
  7. CRITICAL: Turn off proxy (set Proxy status to DNS only / Gray Cloud) so that we can verify and issue SSL.
  8. Click Save. Allow up to 1-5 minutes for propagation.
  1. Log in to your GoDaddy Domain Control Center.
  2. Select your domain name to access the Domain Settings page.
  3. Scroll down to Additional Settings and click Manage DNS.
  4. Click Add at the bottom of the records table.
  5. Select CNAME from the Type dropdown.
  6. Set Host to your desired subdomain (e.g., seo).
  7. Set Points to to localseogen.com.
  8. Leave TTL as default (e.g., 1 Hour) and click Save.
  1. Log in to your Namecheap Dashboard.
  2. Select Domain List from the left sidebar and click Manage next to your domain.
  3. Go to the Advanced DNS tab at the top.
  4. In the Host Records section, click Add New Record.
  5. Choose CNAME Record as the Type.
  6. Set Host to your desired subdomain (e.g., seo).
  7. Set Value to localseogen.com.
  8. Set TTL to Automatic and click the Green Checkmark to save.
Visitors to the root of your custom domain (e.g. seo.mybusiness.com/) will be redirected here.
