# HELP-REQUEST.md

## Database Setup Request

I require a PostgreSQL database to proceed with key features like customer authentication, usage-based pricing, and expanded payment options (e.g., PayPal integration).

Please set up a PostgreSQL database and provide the connection string as a Vercel environment variable.

**Database Service Preference:** Vercel Postgres (due to existing Vercel deployment)

**Required Schema:**

*   **`users` table:**
    *   `id` (Primary Key, UUID or SERIAL)
    *   `email` (UNIQUE, VARCHAR(255))
    *   `password_hash` (VARCHAR(255))
    *   `created_at` (TIMESTAMP WITH TIME ZONE, DEFAULT CURRENT_TIMESTAMP)
    *   `updated_at` (TIMESTAMP WITH TIME ZONE, DEFAULT CURRENT_TIMESTAMP)
*   **`agencies` table:**
    *   `id` (Primary Key, UUID or SERIAL)
    *   `user_id` (Foreign Key to `users.id`)
    *   `name` (VARCHAR(255))
    *   `created_at` (TIMESTAMP WITH TIME ZONE, DEFAULT CURRENT_TIMESTAMP)
    *   `updated_at` (TIMESTAMP WITH TIME ZONE, DEFAULT CURRENT_TIMESTAMP)
*   **`clients` table:**
    *   `id` (Primary Key, UUID or SERIAL)
    *   `agency_id` (Foreign Key to `agencies.id`)
    *   `name` (VARCHAR(255))
    *   `created_at` (TIMESTAMP WITH TIME ZONE, DEFAULT CURRENT_TIMESTAMP)
    *   `updated_at` (TIMESTAMP WITH TIME ZONE, DEFAULT CURRENT_TIMESTAMP)
*   **`page_credits` table:**
    *   `id` (Primary Key, UUID or SERIAL)
    *   `user_id` (Foreign Key to `users.id`)
    *   `credits_remaining` (INTEGER, DEFAULT 0)
    *   `created_at` (TIMESTAMP WITH TIME ZONE, DEFAULT CURRENT_TIMESTAMP)
    *   `updated_at` (TIMESTAMP WITH TIME ZONE, DEFAULT CURRENT_TIMESTAMP)

**Environment Variable:**
Please provide the database connection string as a Vercel environment variable, for example, `POSTGRES_PRISMA_URL` or `DATABASE_URL`.
